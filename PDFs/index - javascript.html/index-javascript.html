Clase Interactiva de Fundamentos de JavaScript

Este documento demuestra los conceptos clave de JavaScript. Cada sección tiene una explicación teórica seguida de un ejemplo práctico para experimentar.

1. Variables y Tipos de Datos

Las variables son "contenedores" para almacenar datos. En JavaScript moderno, usamos let para variables que pueden cambiar de valor y const para valores constantes que no se pueden reasignar.

JavaScript tiene tipado dinámico, lo que significa que una variable puede cambiar de tipo durante la ejecución. Para saber el tipo de una variable, usamos el operador typeof.

Demostración Práctica

Ingresa un valor en el campo de texto y observa cómo JavaScript identifica su tipo.

const miConstante = "Valor fijo"; let miVariable = document.getElementById('tipoDatoInput').value; let tipo = typeof miVariable; document.getElementById('tipoDatoOutput').textContent = `El tipo de dato es: ${tipo}`;

2. Operadores

Los operadores nos permiten realizar operaciones con variables y valores.

  • Aritméticos: Para cálculos matemáticos (+, -, *, /, %).
  • De Comparación: Devuelven true o false. Es crucial usar la igualdad estricta === que compara valor Y tipo.
  • Lógicos: Para combinar condiciones (&& AND, || OR, ! NOT).

Demostración Práctica

Compara dos números para ver los operadores en acción.

3. Estructuras de Control

Nos permiten modificar el flujo de ejecución de nuestro programa.

  • Condicionales (if/else, switch): Ejecutan código si se cumple una condición. switch es útil cuando se compara una misma variable con múltiples valores exactos.
  • Bucles (for, while): Repiten un bloque de código. El bucle for es ideal cuando sabes el número exacto de iteraciones. El bucle for...of itera sobre valores de objetos iterables como los arrays.

    4. Funciones

    Son bloques de código reutilizables que realizan una tarea específica. Ayudan a organizar y evitar repetir código.

    Las funciones flecha (introducidas en ES6) son una forma más concisa de escribir funciones. Son muy útiles pero no pueden ser usadas como constructores.

    5. DOM, Eventos y Manipulación

    El DOM (Document Object Model) es una representación de nuestro HTML que JavaScript puede manipular. Podemos seleccionar elementos, cambiar su contenido, estilo y atributos.

    Los eventos son acciones que ocurren en el navegador (clics, pulsaciones de teclas, etc.). Usamos addEventListener para "escuchar" estos eventos y ejecutar una función en respuesta.

    Para crear y eliminar elementos dinámicamente, usamos métodos como document.createElement(), appendChild() y remove().

    La delegación de eventos es una técnica eficiente donde se asigna un solo "listener" a un elemento padre para manejar eventos en múltiples hijos, incluso los que se añaden después.
